These are hand held range finders. I have Included the Autodesk Fusion Files Incase you would like to modify them. I plan on making my deer range finder with a hinge, so i can swing it out of the way. 

THE MATH HERE IS APPROXIMATE! These range finders were designed with the average height of a white tail deer and how tall they appear at distances. 

EXAMPLE: if a deer is 38" tall at 0 yards away it will be a tenth (1/10th) of the height at 10 yards away. 38/10 = 3.8" tall perspective.
- This will not give you superior accuracy. All values are within a allowable range; +- 1 Yard. Depending on how tall the animal is (On Average). See NOTES.png for my work.
